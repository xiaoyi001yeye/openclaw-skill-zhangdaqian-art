# 艺术品抓取Skill使用指南

## 功能说明
1. **关键字抓取**：输入关键字，从雅昌艺术网拍卖搜索页抓取所有相关艺术品数据，包括列表页基础信息和详情页完整信息，每个艺术品保存为独立JSON文件
2. **列表查询**：查询所有已抓取的艺术品名称列表
3. **详情查询**：输入艺术品名称，返回对应的完整JSON数据文件

## 目录结构
```
art-crawler/
├── SKILL.md                  # 本文件
├── scripts/
│   └── crawler.py            # 核心爬虫脚本
└── data/                     # 抓取结果存储目录（自动创建）
```

## 使用方法

### 1. 关键字抓取
当用户需要抓取某关键字的艺术品数据时，执行：
```bash
python scripts/crawler.py crawl "关键字"
```
- 自动抓取所有分页的列表数据
- 自动进入每个艺术品详情页抓取完整信息
- 每个艺术品保存为`data/[艺术品ID].json`文件
- 自动去重，已抓取的艺术品不会重复抓取
- **断点续抓**：任务中断后重新执行会从上次进度继续
- **实时进度上报**：每处理完一页自动上报当前进度（页码、已抓取数量、最后更新时间）
- **异常中断告警**：如果任务被中断或意外停止，会自动保存进度并提示当前状态

### 2. 查看抓取进度
当用户需要了解当前抓取任务状态时，执行：
```bash
python scripts/crawler.py progress
```
- 显示当前任务状态（运行中/已中断/已停止/已完成）
- 显示当前抓取页码、已抓取总数、最后更新时间
- 随时可查询，无需等待任务完成

### 3. 查询已抓取列表
当用户询问已抓取哪些艺术品时，执行：
```bash
python scripts/crawler.py list
```
- 返回所有已抓取艺术品的名称列表
- 按抓取时间倒序排列

### 4. 查询艺术品详情
当用户需要某艺术品的完整数据时，执行：
```bash
python scripts/crawler.py get "艺术品名称"
```
- 支持模糊匹配，返回最匹配的艺术品JSON数据
- 如果有多个匹配结果，返回所有匹配项

## 注意事项
- 抓取间隔默认设置为2秒，避免对目标网站造成压力
- 数据自动保存在`data/`目录下，永久存储
- 支持断点续抓，中断后重新执行会从上次进度继续
- 进度自动保存在`crawl_progress.json`文件中，可随时通过`progress`命令查询
- 意外中断时会自动保存当前进度，下次启动自动恢复
